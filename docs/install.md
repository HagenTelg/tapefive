# Install

```bash
pip install tapefive
# for contributors:
pip install -e .[docs]
```

## External prerequisites

LBLRTM & LNFL (Atmospheric & Environmental Research): see AER’s official pages and GitHub.

LBLRTM: https://rtweb.aer.com/

AER-RC GitHub: LBLRTM, LNFL, AER_Line_File

AER Line File (TAPE1 → TAPE3 via LNFL)

tapefive assumes lnfl and lblrtm are installed and discoverable on PATH.
